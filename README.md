# MooColor

A simple color library for javascript.

## Install

``` sh
npm install moo-color
```

## Usage

``` js
import { MooColor } from 'moo-color';

const color = new MooColor('#f00');

color.whiten(10).toString('hwb'); // returns 'hwb(0, 10%, 0)'
```
